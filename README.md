# 📱 Financial Receipt Scanner

An AI-powered web application that automatically extracts, analyzes, and manages financial transactions from receipts and invoices using OCR (Optical Character Recognition) technology.

## 🌐 Live Demo

**Try the application here:**

🔗 https://artifacts-mauve.vercel.app/

## 📖 Overview

Financial Receipt Scanner helps users digitize and organize their financial records by simply uploading receipt images. The application automatically extracts transaction details, stores them securely, and provides personalized financial insights based on spending patterns.

This project was built to explore OCR technology, modern web development practices, and financial data management using a scalable cloud-based architecture.

## ✨ Key Features

* 📸 Smart Receipt Scanning with OCR
* 🧾 Automatic Transaction Extraction
* 💰 Financial Insights and Recommendations
* ☁️ Cloud Database Integration with Supabase
* 📊 Transaction History Management
* 🎨 Modern Responsive User Interface
* 🔒 Secure Environment Variable Management
* ⚡ Fast Performance with Next.js

## 🛠️ Technology Stack

| Category          | Technology              |
| ----------------- | ----------------------- |
| Frontend          | Next.js 16, React 19    |
| Language          | TypeScript              |
| Styling           | Tailwind CSS 4          |
| Database          | Supabase (PostgreSQL)   |
| OCR               | Tesseract.js            |
| Icons             | Heroicons, Lucide React |
| Deployment        | Vercel                  |
| Development Tools | ESLint, Node.js         |

## 🎯 Project Goals

* Automate manual receipt recording
* Improve personal financial management
* Demonstrate OCR implementation in a real-world application
* Explore cloud-native full-stack development
